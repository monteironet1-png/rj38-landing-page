# RJ38 GROUP — Coming Soon Landing Page

Futuristic coming soon landing page for [rj38.com](https://rj38.com).

## Stack
- Pure HTML + CSS + Vanilla JS
- No frameworks, no dependencies
- Google Fonts: Orbitron + Inter

## Files
- `index.html` — Main page
- `style.css` — Styles and animations
- `script.js` — Starfield canvas animation

## Deploy on Hostinger
1. Download all 3 files
2. Go to Hostinger → File Manager
3. Upload to `public_html` of rj38.com
4. Done!
